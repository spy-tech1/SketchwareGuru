Tutorial: How to Create a Battery Percentage App

📹 Watch Video

📖 Introduction

This tutorial will walk you through how to create a simple battery percentage app using Sketchware Pro. You’ll learn how to add source code that shows battery percentage of a device

📟 Source Code Used

BatteryManager bm = (BatteryManager) getSystemService(BATTERY_SERVICE);

int battery_percent = bm.getIntProperty(BatteryManager.BATTERY_PROPERTY_CAPACITY);

👉𝗧𝗲𝘅𝘁𝘃𝗶𝗲𝘄𝗜𝗗👈.setText(battery_percent + "%");

🧩 What You'll Learn

  • How to use "add source directly" block

🛠️ Steps

  1. Create a new project in Sketchware Pro.
  2. Add a two textview widgets
  3. Add a source code directly block
  4. Paste the source code above and input the correct Textview ID in the "👉TextviewID👈" e.g. Textview2.setText(battery_percent + "%");
  5. Run the app and test it
